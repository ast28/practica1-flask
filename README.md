### Descripció aplicació
Aquesta aplicació web, creada amb el framework Flask, permet als usuaris introduir noms mitjançant un formulari interactiu. 
El sistema processa aquestes dades i les emmagatzema en una base de dades SQLite, 
la qual cosa garanteix que la informació es mantingui guardada encara que es tanqui. 

### Instruccions per executar en local
#### 1r. Crear i activar un entorn virtual de python: 

    python3 -m venv venv

    source venv/bin/activate

#### 2n. Instal·lar les dependències: 

    pip install -r requirements.txt


#### 3r. Executar el servidor de la aplicació: 

    python app.py

#### 4t. Accedir a la pàgina web: 

    http://localhost:8000
